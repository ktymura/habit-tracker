# Habit Tracker Frontend

Frontend dla projektu Habit Tracker przygotowany w oparciu o:

- React
- Vite
- Tailwind CSS
- ESLint
- Prettier

## Start

```bash
npm install
npm run dev
```

## Skrypty

```bash
npm run dev
npm run build
npm run lint
npm run format:check
```

## Struktura `src`

- `app/` - punkt startowy aplikacji i glowny komponent
- `components/` - wspolne komponenty UI
- `features/` - logika domenowa kolejnych funkcjonalnosci
- `layouts/` - wspolne uklady stron
- `lib/` - pomocnicze utilsy i konfiguracja
- `mocks/` - mockowe dane i adaptery
- `pages/` - ekrany aplikacji
- `services/` - klient API i warstwa komunikacji
- `styles/` - style globalne
- `types/` - wspolne typy
